📋 Request Flow — Numbers on diagram

① ALB
Receives HTTP request from user browser. Routes to healthy EC2 instance.
② EC2
Instance launched by EB with Immutable policy. Port 22 closed. SSM Agent running.
③ Flask App
At startup reads 7 SSM params via IAM role. Assembles DB URL in memory. Never stores secrets.
④ RDS
PostgreSQL inside same VPC. Accepts connections only from EB security group on port 5432.
⑤ SSM Store
Holds all secrets encrypted with KMS. Only EC2 IAM role can read /myapp/* parameters.
